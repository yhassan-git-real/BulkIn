# BulkIn Desktop - Keyboard Shortcuts

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl+1** | Switch to Processing | Navigate to the Processing tab |
| **Ctrl+2** | Switch to Settings | Navigate to the Settings tab |
| **Ctrl+3** | Switch to Logs | Navigate to the Logs tab |
| **F1** | Show About | Display application information, version, and features |

## Processing View

| Shortcut | Action | Description |
|----------|--------|-------------|
| **F5** | Start Processing | Begin processing all files in the queue |
| **Ctrl+P** | Pause Processing | Pause the current processing operation |

## Settings View

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl+S** | Save Settings | Save all configuration changes to appsettings.json |

## Navigation Tips

- Use **Tab** to navigate between input fields
- Use **Enter** to activate focused buttons
- Use **Escape** to close dialogs
- All shortcuts are displayed in tooltips when hovering over buttons

## Quick Access

The status bar at the bottom of the window displays commonly used shortcuts for quick reference:
- `Ctrl+1/2/3`: Switch tabs
- `F5`: Start processing
- `Ctrl+S`: Save settings
- `F1`: Help
