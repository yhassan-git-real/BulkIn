🚀 QUICK START - Three Easy Steps:
═══════════════════════════════════════════════════════════

Step 1: Create Desktop Shortcut (One-Time)
   ➜ Double-click: CreateDesktopShortcut.bat
   ➜ A "BulkIn" icon will appear on your desktop

Step 2: Configure Database Settings
   ➜ Edit: src\BulkIn\appsettings.json
   ➜ Set your SQL Server name and database name

Step 3: Run BulkIn
   ➜ Double-click the "BulkIn" icon on your desktop
   ➜ Or double-click: BulkIn-Menu.bat

⚙️ CONFIGURATION FILE:
═══════════════════════════════════════════════════════════

Location: src\BulkIn\appsettings.json

Required Settings:
  • ServerName        → Your SQL Server instance
  • DatabaseName      → Target database
  • SourceFilePath    → Folder with text files
